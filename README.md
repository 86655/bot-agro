# Agro Bot 

Projeto simples desenvolvido para estudar programação, Git e GitHub.

## Tecnologias usadas

- JavaScript
- Node.js
- Git
- GitHub

## Como rodar o projeto

1. Instalar dependências

npm install

2. Executar o projeto

node teste.js

## Objetivo

Criar um bot que futuramente possa ajudar com informações úteis para o agronegócio, como:

- previsão do tempo
- preços de commodities
- dados agrícolas
